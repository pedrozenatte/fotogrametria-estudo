### 12 - Write a python class that estimate the camera pose given a planar marker, using homographies. 

**Vamos entender a diferença do que fizemos anteriormente para estimar a pose da câmera:**
Quando usamos o método de pontos 3D e 2D para estimar a pose da câmera, estamos fazendo a correspondência entre pontos 3D do espaço e seus respectivos pontos projetados na imagem 2D, e, para isso, precisamos conhecer tantos os pontos 3D, com coordenadas reais, como os pontos 2D correspondentes.
Agora, quando vamos utilizar a estimativa de pose com homografia, estamos utilizando um marcador plano, ou seja, os pontos 3D na verdade podem ser simplificados para pontos 2D, uma vez que não há profundidade envolvida. Dessa forma, podemos projetar os pontos 2D da imagem para outro plano 2D, que faz referencia ao 3D. Só vai funcionar com superfícies planas.
