### 11 - What is a Homography Matrix, how you can use that?

É uma matriz 3x3 com 8 graus de liberdade que descreve como os pontos de uma imagem mudam para outra, ou seja, se os pontos estão em uma superfície plana, como um tabuleiro, a homografia descreve como esses pontos mudam de uma imagem para outra. Em outras palavras, a matriz descreve a transformação de pontos entre imagens, assumindo que esses pontos estão em uma superfície plana. 
Basicamente estamos transformando pontos de um plano para outro plano. 
Uma das formas que podemos usar essa matriz é com o matching de imagens para projetar os pontos da primeira imagem na segunda imagem usando homografia.
